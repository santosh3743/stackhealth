# Disposable / throwaway email domains. One domain per line, lowercase.
# Lines starting with '#' and blank lines are ignored.
#
# Curated from the disposable-email-domains project
# (https://github.com/disposable-email-domains/disposable-email-domains)
# — a focused subset of the most common providers. We deliberately keep
# this list small and curated rather than vendoring all ~3,000 entries,
# because:
#   1. The long tail is rarely worth the binary size + match cost.
#   2. False positives on a real customer email are worse than letting
#      one obscure disposable through.
#
# When you find a real throwaway domain in the subscribers table that
# isn't here, just add the apex domain (one line) and ship — no other
# wiring required.
